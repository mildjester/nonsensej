Initial Setup for Ubuntu 20.10

posted: 2021-02-11

ubuntu

Environment

  • Ubuntu 20.10
  • ThinkPad T480

These are my notes from installing Ubuntu 20.10 on a Thinkpad T480.

Japanese Input Settings

Japanese input is possible by default, but since you have to switch the input source to Mozc,
I will increase the priority of the Mozc input source.

Open “Settings” from the application launcher, and then open “Region & Language”.
You will see “Japanese” and “Japanese (Mozc)” listed under Input Sources.
Rearrange them so that “Japanese (Mozc)” is at the top.

You can change the order by dragging the six dots on the left side of each input source.

Making IME ON/OFF Mac-like

Open “Mozc Settings” from the application launcher, and then open the “General” tab.
Click the “Edit” button next to “Select keymap” to change the key settings.

Assign “IME On” to Henkan/Hiragana/Katakana, and
assign “IME Off” to Muhenkan.

Now you can switch IME ON/OFF like on a Mac.

Remapping Caps Lock to Ctrl and Hankaku/Zenkaku to Esc

I will use Gnome Tweak Tool for this.
First, install it with the following command:

sudo apt install gnome-tweak-tool

You can launch it from the launcher, so open “Utilities > Tweaks”.
Once Tweaks is open, click “Additional Layout Options” under “Keyboard & Mouse”.

When the options screen opens, check “Ctrl position > Caps Lock is Ctrl” and “Japanese keyboard options > Make Zenkaku Hankaku an additional Esc”.

Disabling Tiling Functionality

By default, windows automatically maximize when dragged to the edge of the screen.
Since I don’t need this feature, I will disable it with the following command:

gsettings set org.gnome.mutter edge-tiling false

Moving the Status Bar to the Bottom

By default, the bar displaying the clock and battery level is at the top of the window, and the application bar is on the left side of the window.
Since I prefer having everything aligned at the bottom of the window like in Windows, I will use Dash to Panel to move it.

To install Dash to Panel, simply click the On/Off switch on the following page to turn it On:
https://extensions.gnome.org/extension/1160/dash-to-panel/

This will move the status bar to the bottom of the window.
If you want to change the settings, right-click on the status bar and select “Dash to Panel Settings”.

Installing Various Software

Basic Set

sudo apt install vim curl npm

PHP

sudo apt install php7.4 composer php7.4-bcmath php7.4-common php7.4-gd php7.4-json php7.4-mbstring php7.4-mysql php7.4-xml php7.4-xmlrpc

MySQL Workbench

I downloaded the deb file from the official page and installed it.
https://dev.mysql.com/downloads/workbench/

Shutter (Screenshot Tool)

Useful because it allows you to write text and arrows on existing images.

sudo add-apt-repository ppa:linuxuprising/shutter
sudo apt update
sudo apt install shutter

Preventing Apache from Starting

If you are using Docker etc. for your testing environment, a local Apache server can be a nuisance, so I will stop it.

systemctl stop apache2
systemctl disable apache2
