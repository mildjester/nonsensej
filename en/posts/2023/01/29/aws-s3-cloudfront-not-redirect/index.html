【AWS】S3 Redirect Not Working

posted: 2023-01-29

AWS

Issue Encountered

I was publishing a webpage using AWS S3 static website hosting + CloudFront.
I needed to implement a redirect on this webpage, so I configured the redirect settings in S3 based on the following documentation:
AWS Official S3 Documentation

However, when I actually accessed the page from a browser, the redirect did not occur.
Clearing the CloudFront cache was also ineffective, and the redirect settings seemed correct upon review.

When I tried opening the bucket website endpoint URL directly, it redirected correctly.
Therefore, the issue likely lies in the integration with CloudFront.

Cause

It seems that if you select S3 from the options when specifying the origin domain in CloudFront, the redirects configured in S3 will not work.
To make the redirect settings effective, instead of “selecting” S3, you need to directly enter the domain of the bucket website endpoint.

By doing the above, the redirect started working even when accessed from a browser.
